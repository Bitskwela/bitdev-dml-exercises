# Chapter 6 — Quiz Answer Key (with Questions)

Source: `quiz.md`

1. Q1 — In a form-based project like a calculator, what is the 'submit' event used for?

   - Answer: **B** — The event that fires when a form is submitted

2. Q2 — Which method prevents a form from submitting and reloading the page?

   - Answer: **A** — `event.preventDefault()`

3. Q3 — In a mini project, you can use querySelector to select form input elements.

   - Answer: **True**

4. Q4 — Which of these can be used to get or set content in a DOM element?

   - Answer: **D** — All of the above

5. Q5 — The .value property is used to get text content from div elements.

   - Answer: **False**

6. Q6 — How do you hide an element in JavaScript without removing it from the DOM?

   - Answer: **C** — `element.style.display = 'none'`

7. Q7 — Which method both accesses AND removes the last element of an array?

   - Answer: **B** — `array.pop()`

8. Q8 — The .push() method adds elements to the end of an array.

   - Answer: **True**

9. Q9 — Which function converts a string to a decimal number in a calculator project?

   - Answer: **A** — `parseFloat()`

10. Q10 — parseInt() will always return a whole number, even if the string contains decimal points.

    - Answer: **False**

11. Q11 — In a POS calculator project, what is the purpose of using an object to store item data?

    - Answer: **C** — It stores key-value pairs for item names and prices

12. Q12 — Which method returns an array of all values in an object?

    - Answer: **B** — `Object.values()`

13. Q13 — You can calculate the sum of array elements using the reduce() method.

    - Answer: **True**

14. Q14 — How do you add content to an element without replacing existing content?

    - Answer: **A** — `element.innerHTML += newContent`

15. Q15 — The 'change' event fires on an input element every time a keystroke occurs.

    - Answer: **False**

16. Q16 — What is required when handling form submissions in a mini project?

    - Answer: **C** — All of the above

17. Q17 — In a calculator project, what validation might you perform on user inputs?

    - Answer: **B** — Check if the input is a valid number

18. Q18 — Conditional statements (if/else) are essential for validating and processing data in mini projects.

    - Answer: **True**

19. Q19 — Which array method is best for finding items that match a specific criteria in a project?

    - Answer: **A** — `filter()`

20. Q20 — You can use template literals to dynamically create HTML content in a mini project.

    - Answer: **True**

21. Q21 — What is the primary purpose of using arrays in a tracking or logging project?

    - Answer: **C** — To store data and calculate totals

22. Q22 — What is the modern way to attach a click event handler to an element?

    - Answer: **B** — `element.addEventListener('click', function)`

23. Q23 — Using inline HTML event attributes (like onclick) is the best practice in modern JavaScript projects.

    - Answer: **False**

24. Q24 — What does the reduce() method do in JavaScript?

    - Answer: **A** — It accumulates a single value from all array elements

25. Q25 — When should you update the DOM in a mini project?

    - Answer: **D** — All of the above

26. Q26 — In a job matching portal, what would be the primary use of filter()?

    - Answer: **B** — Matching or comparing items based on criteria

27. Q27 — LocalStorage can be used to persist data across browser sessions in a mini project.

    - Answer: **True**

28. Q28 — What is basic validation for a required input field in a form?

    - Answer: **A** — Check if the required field is not empty

29. Q29 — An **\_\_** is attached to a DOM element to detect and respond to user interactions like clicks.

    - Answer: **event listener**

30. Q30 — **\_\_** is the process of changing, adding, or removing elements and content from the HTML document using JavaScript.
    - Answer: **DOM manipulation**
