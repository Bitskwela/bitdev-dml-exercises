# Chapter 8 — Quiz Answer Key (with Questions)

Source: `quiz.md`

1. Q1 — What is C++?

   - Answer: **B** — A compiled, general-purpose programming language

2. Q2 — Which of the following is a valid C++ data type?

   - Answer: **C** — `int`

3. Q3 — Variables in C++ must be declared before they are used.

   - Answer: **True**

4. Q4 — What is the correct way to include the standard input/output library in C++?

   - Answer: **A** — `#include <iostream>`

5. Q5 — Which operator is used to read input from the user in C++?

   - Answer: **B** — `cin`

6. Q6 — Which command is used to output data to the console in C++?

   - Answer: **A** — `cout`

7. Q7 — C++ is a dynamically typed language where variables don't need explicit type declarations.

   - Answer: **False**

8. Q8 — Which of these are valid arithmetic operators in C++?

   - Answer: **D** — All of the above except exponentiation

9. Q9 — Which operator is used for assignment in C++?

   - Answer: **B** — `=`

10. Q10 — Which are valid conditional statements in C++?

    - Answer: **A** — `if, else, else if`

11. Q11 — A for loop can be used to repeat a block of code a specific number of times.

    - Answer: **True**

12. Q12 — What does a while loop do?

    - Answer: **C** — Executes until a condition is true

13. Q13 — What does the || operator represent?

    - Answer: **B** — Logical OR

14. Q14 — The && operator returns true if either operand is true.

    - Answer: **False**

15. Q15 — What is a function?

    - Answer: **A** — A reusable block of code that performs a specific task

16. Q16 — Which is a valid function declaration in C++?

    - Answer: **B** — `void functionName() { }`

17. Q17 — What is a pointer?

    - Answer: **C** — A variable that points to a memory location

18. Q18 — Which operator is used to dereference a pointer?

    - Answer: **A** — `*pointer`

19. Q19 — An array is a collection of elements of the same data type stored in contiguous memory.

    - Answer: **True**

20. Q20 — What is the correct way to declare an array in C++?

    - Answer: **B** — `int array[10];`

21. Q21 — What is the starting index of an array in C++?

    - Answer: **A** — `0`

22. Q22 — What is a string in C++?

    - Answer: **B** — A sequence of characters (string)

23. Q23 — The scope of a variable determines where it can be accessed in a program.

    - Answer: **True**

24. Q24 — What is the scope of a variable declared outside all functions?

    - Answer: **A** — Global scope

25. Q25 — What is dynamic memory allocation?

    - Answer: **C** — Allocating memory at runtime (dynamic allocation)

26. Q26 — Which is the correct way to dynamically allocate memory in C++?

    - Answer: **B** — (uses `new` or appropriate API shown in quiz)

27. Q27 — How do you deallocate dynamically allocated memory in C++?

    - Answer: **A** — `delete ptr;`

28. Q28 — What is a struct?

    - Answer: **C** — A composite data type with multiple fields

29. Q29 — A class is similar to a struct but provides encapsulation and access control.

    - Answer: **True**

30. Q30 — What is a constructor?

    - Answer: **B** — A special function that initializes a new object of a class

31. Q31 — What are the three access modifiers in C++ classes?

    - Answer: **A** — `public, private, protected`

32. Q32 — What is inheritance?

    - Answer: **C** — Creating a new class based on an existing class (reuse/extend)

33. Q33 — Polymorphism allows objects of different classes to be treated through the same interface.

    - Answer: **True**

34. Q34 — What is a virtual function?

    - Answer: **B** — A function that can be overridden in derived classes

35. Q35 — What are templates used for?

    - Answer: **A** — Write functions/classes that work with any data type

36. Q36 — What is the correct syntax for exception handling in C++?

    - Answer: **C** — `try { } catch(...) { }`

37. Q37 — What is the STL?

    - Answer: **B** — Standard Template Library (collection of containers, algorithms)

38. Q38 — A vector in STL is a fixed-size array that cannot be resized after creation.

    - Answer: **False**

39. Q39 — Pointers store the **\_\_** address of a variable.

    - Answer: **memory**

40. Q40 — C++ is an **\_\_** programming language that supports classes and inheritance.
    - Answer: **object-oriented**
