# Chapter 5 — Quiz Answer Key (with Questions)

Source: `quiz.md`

1. Q1 — What is a variable in JavaScript?

   - Answer: **B** — A named space in memory that stores a value

2. Q2 — Which keyword has function scope and can be re-declared in the same scope?

   - Answer: **A** — `var`

3. Q3 — Which variable declaration keyword cannot be reassigned after initialization?

   - Answer: **B** — `const`

4. Q4 — Strings, numbers, and booleans are primitive data types in JavaScript.

   - Answer: **True**

5. Q5 — What scope do 'let' and 'const' have in JavaScript?

   - Answer: **C** — Block scope

6. Q6 — In JavaScript, typeof null returns 'null'.

   - Answer: **False**

7. Q7 — Which of the following is a primitive data type in JavaScript?

   - Answer: **D** — All of the above

8. Q8 — A function is a reusable block of code that performs a specific task.

   - Answer: **True**

9. Q9 — What is the correct syntax to declare a function in JavaScript?

   - Answer: **B** — `function myFunc() { }`

10. Q10 — Parameters and arguments are the same thing in JavaScript.

    - Answer: **False**

11. Q11 — What is the syntax for an arrow function in JavaScript?

    - Answer: **A** — `=>`

12. Q12 — What is the correct syntax for array destructuring?

    - Answer: **B** — `const [a, b, c] = array`

13. Q13 — What is the spread operator (...) used for in JavaScript?

    - Answer: **D** — A way to spread elements of an array or properties of an object

14. Q14 — The 'map()' method modifies the original array in place.

    - Answer: **False**

15. Q15 — Which loop iterates over the values of an iterable object?

    - Answer: **C** — `for...of`

16. Q16 — What is a higher-order function?

    - Answer: **A** — A function that returns a function

17. Q17 — What does the 'filter()' method do?

    - Answer: **C** — Returns elements that pass a condition

18. Q18 — Arrow functions have a lexical 'this' binding, unlike regular functions.

    - Answer: **True**

19. Q19 — Which array method accumulates a value by iterating through all elements?

    - Answer: **A** — `reduce()`

20. Q20 — Objects in JavaScript are mutable reference types.

    - Answer: **True**

21. Q21 — What is the syntax feature that allows you to unpack values from arrays or objects into variables?

    - Answer: **A** — Destructuring

22. Q22 — Which method selects a DOM element by its ID?

    - Answer: **B** — `getElementById`

23. Q23 — Event listeners can be added to DOM elements using addEventListener().

    - Answer: **True**

24. Q24 — What can the DOM API be used for?

    - Answer: **D** — All of the above

25. Q25 — Which keyword defines the constructor method in a JavaScript class?

    - Answer: **C** — `constructor()`

26. Q26 — Which keyword is used for class inheritance in JavaScript?

    - Answer: **A** — `extends`

27. Q27 — In JavaScript, '===' performs type coercion before comparison.

    - Answer: **False**

28. Q28 — Which function is used to make HTTP requests in JavaScript?

    - Answer: **A** — `fetch()`

29. Q29 — Which method is used to parse JSON from a fetch response?

    - Answer: **C** — `.json()`

30. Q30 — Which operator checks if an object is an instance of a class?

    - Answer: **B** — `instanceof`

31. Q31 — What are the three states of a Promise?

    - Answer: **B** — fulfilled, rejected, pending

32. Q32 — Promises in JavaScript handle asynchronous operations.

    - Answer: **True**

33. Q33 — Which method is used to handle the fulfilled state of a Promise?

    - Answer: **C** — `.then()`

34. Q34 — Which keyword pauses execution until a Promise is resolved?

    - Answer: **A** — `await`

35. Q35 — The 'async' keyword makes a function return a regular value instead of a Promise.

    - Answer: **False**
