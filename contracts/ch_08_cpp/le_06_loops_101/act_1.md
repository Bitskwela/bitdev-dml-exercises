**Time Allotment: 20 minutes**

## Tasks for students

Topics Covered: `for` loop, loop counter, iteration

- Automate the generation of 50 numbered reminder notices:
  - Use a `for` loop to repeat a message 50 times.
  - Define the loop initialization (`int i = 1`), the condition (`i <= 50`), and the update (`i++`).
  - Inside the loop, print the reminder message: "Reminder #[number]: Please segregate your waste properly."
  - Ensure the current value of the loop counter `i` is displayed in each message.
