## Lesson 12 Exercises: String Mastery

---
### 1. f-String Report (6 minutes)
Variables name="Ana", score=87.456. Format: "Name: Ana | Score: 87.46".

---
### 2. Zero-Padded IDs (5 minutes)
Generate list of IDs 1-10 formatted as "ID-00001" to "ID-00010".

---
### 3. Title Case Normalization (5 minutes)
Input: "JUAN dela CRUZ". Output: "Juan Dela Cruz" using title().

---
### 4. CSV Line Parser (7 minutes)
Parse "Ana, 21, Sto. Niño " into list ["Ana","21","Sto. Niño"] stripping spaces.

---
### 5. Column Alignment (8 minutes)
Print table: names left 15, ages right 5. Three rows.

---
### 6. Substring Search (6 minutes)
Function contains_ignore_case(text, substr): use lower() + in.

---
### 7. Reverse Words (7 minutes)
Input: "Hello World". Output: "World Hello" (split, reverse, join).

---
### 8. Validation with isdigit (6 minutes)
Function safe_parse(s): return int if s.isdigit() else None.

---
### 9. Stretch: Template Engine (Optional)
