# Python Programming

Welcome to Python Programming! This course covers Python fundamentals from zero to building real-world applications.

## Course Overview

Python is one of the most popular programming languages in the world, known for its simplicity and versatility. Used by companies like Google, Netflix, Instagram, and Spotify.

## Lessons

1. **Introduction to Python** - What is Python, setup, first program
2. **Variables and Data Types** - Storing and working with data
3. **Operators and Expressions** - Performing operations
4. **Input and Output** - Interacting with users
5. **Control Flow: Conditionals** - Making decisions
6. **Loops** - Repeating actions
7. **Lists** - Working with collections
8. **Functions** - Reusable code blocks
9. **Dictionaries** - Key-value data structures
10. **File Handling** - Reading and writing files

## Learning Path

Each lesson includes:
- Comprehensive material with Tian's story
- Code examples and explanations
- Practical exercises
- Scenario-based quizzes

Estimated time: 8-10 minutes per lesson
