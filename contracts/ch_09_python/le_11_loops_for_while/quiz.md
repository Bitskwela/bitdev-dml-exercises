# Lesson 11 Quiz: Loops (for & while)

---
# Quiz 1
## Scenario: Batch Processing
Tian automates repetitive tasks.

**Question 1:** range(3, 8) produces:
A. [3,4,5,6,7,8]
B. [3,4,5,6,7]
C. [3,8]
D. Error

**Answer:** B
**Explanation:** range(start, stop) excludes stop.

---

**Question 2:** break statement does:
A. Skip current iteration
B. Exit loop immediately
C. Pause execution
D. Reset counter

**Answer:** B
**Explanation:** break exits loop.

---

**Question 3:** continue effect:
A. Exit loop
B. Skip rest of current iteration
C. Increment by 2
D. Reverse direction

**Answer:** B
**Explanation:** continue jumps to next iteration.

---

**Question 4:** Loop else clause runs when:
A. Break executed
B. Loop completes normally without break
C. Error occurs
D. Always

**Answer:** B
**Explanation:** else runs if no break encountered.

---

**Question 5:** enumerate() provides:
A. Only values
B. Index and value pairs
C. Reversed iteration
D. Random access

**Answer:** B
**Explanation:** enumerate yields (index, value).

# Quiz 2
## Scenario: Optimization Review
Rhea Joy checks loop efficiency.

**Question 6:** Infinite loop risk:
A. for loop over list
B. while True without break
C. range with step
D. enumerate usage

**Answer:** B
**Explanation:** while True needs explicit break.

---

**Question 7:** zip([1,2],["a","b"]) yields:
A. [(1,"a"),(2,"b")]
B. [1,2,"a","b"]
C. {1:"a",2:"b"}
D. Error

**Answer:** A
**Explanation:** zip pairs elements.

---

**Question 8:** Iterating dict best uses:
A. for key in d:
B. for i in range(len(d)):
C. while d:
D. d.loop()

**Answer:** A
**Explanation:** Direct key iteration standard.

---

**Question 9:** List comprehension advantage:
A. Slower than loop
B. More verbose
C. Concise, often faster
D. Only works on strings

**Answer:** C
**Explanation:** Comprehensions concise & optimized.

---

**Question 10:** Nested loop complexity:
A. O(n)
B. O(n + m)
C. O(n * m)
D. O(log n)

**Answer:** C
**Explanation:** Nested loops multiply iterations.
