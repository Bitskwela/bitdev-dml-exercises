# Quiz 2
## Scenario: Optimization Review
Rhea Joy checks loop efficiency.

**Question 6:** Infinite loop risk:
A. for loop over list
B. while True without break
C. range with step
D. enumerate usage

**Question 7:** zip([1,2],["a","b"]) yields:
A. [(1,"a"),(2,"b")]
B. [1,2,"a","b"]
C. {1:"a",2:"b"}
D. Error

**Question 8:** Iterating dict best uses:
A. for key in d:
B. for i in range(len(d)):
C. while d:
D. d.loop()

**Question 9:** List comprehension advantage:
A. Slower than loop
B. More verbose
C. Concise, often faster
D. Only works on strings

**Question 10:** Nested loop complexity:
A. O(n)
B. O(n + m)
C. O(n * m)
D. O(log n)

---
## Answers
1: B  
2: B  
3: B  
4: B  
5: B  
6: B  
7: A  
8: A  
9: C  
10: C  

---
## Detailed Explanations
Q1 range(start, stop) excludes stop.  
Q2 break exits loop.  
Q3 continue jumps to next iteration.  
Q4 else runs if no break encountered.  
Q5 enumerate yields (index, value).  
Q6 while True needs explicit break.  
Q7 zip pairs elements.  
Q8 Direct key iteration standard.  
Q9 Comprehensions concise & optimized.  
Q10 Nested loops multiply iterations.
