# Quiz 1
## Scenario: Formatting Rescue
Tian pastes code with mixed tabs/spaces. Identify corrections.

**Question 1:** Python uses indentation to define:
A. Variable scope only
B. Code blocks / suites
C. Memory allocation
D. Runtime optimization hints

**Answer:** B  
**Explanation:** Indentation marks suites (blocks).

---

**Question 2:** Proper block opener requires:
A. Parentheses at end
B. Colon followed by indented block
C. Semicolon
D. Trailing comma

**Answer:** B  
**Explanation:** Colon signals following block.

---

**Question 3:** Best place for explaining a function's purpose:
A. Inline trailing comment
B. External text file
C. Function docstring
D. Above module import

**Answer:** C  
**Explanation:** Docstring stays with function, discoverable.

---

**Question 4:** Mixed tabs/spaces risk:
A. Automatic correction
B. Silent different logic paths
C. TabError / inconsistent indentation
D. Performance drop only

**Answer:** C  
**Explanation:** Mixing leads to errors/confusion.

---

**Question 5:** Redundant comment example:
A. # compute normalized average with non-obvious math
B. # i = i + 1 directly above i += 1
C. # TODO: handle missing field actionable note
D. # explain algorithm step helpful context

**Answer:** B  
**Explanation:** Comment restates obvious codeâ€”avoid.

# Quiz 2
## Scenario: Style Review
Rhea Joy enforces readability across contributors.

**Question 6:** Naming convention for functions:
A. camelCase
B. snake_case
C. UPPERCASE
D. PascalCase

**Answer:** B  
**Explanation:** snake_case standard for functions.

---

**Question 7:** Line continuation preferred using:
A. Backslash anywhere
B. Parentheses grouping expression
C. Semicolon chaining
D. Double indent

**Answer:** B  
**Explanation:** Parentheses avoid fragile backslashes.

---

**Question 8:** Docstrings are accessible via:
A. info()
B. help()
C. printdoc()
D. manual()

**Answer:** B  
**Explanation:** help() introspects docstrings.

---

**Question 9:** Which improves team readability?
A. Consistent 4-space indentation
B. Random varied indentation width
C. Tabs only mixed randomly
D. Removing all blank lines

**Answer:** A  
**Explanation:** Uniform indentation supports quick parsing.

---

**Question 10:** Purpose of PEP 8 guidelines:
A. Enforce security controls
B. Standardize style for readability
C. Eliminate need for comments
D. Replace testing

**Answer:** B  
**Explanation:** Shared conventions ease collaboration.
