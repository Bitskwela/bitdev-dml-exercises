B. Bell curve

D. Step function

## Detailed Explanations
Q1 Mean sums and divides by count.  
Q2 Median unaffected by extreme values.  
Q3 Std dev quantifies variability.  
Q4 Quartiles split data into four equal parts.  
Q5 IQR * 1.5 rule identifies outliers.  
Q6 Probability = favorable outcomes / total.  
Q7 Independent events multiply probabilities.  
Q8 Conditional: restrict to B occurring.  
Q9 Z-score standardizes position in distribution.  
Q10 Normal curve symmetric bell shape.
