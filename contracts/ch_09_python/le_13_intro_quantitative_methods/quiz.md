# Quiz 1
## Scenario: Budget Forecasting
Tian explores modeling options.

**Question 1:** Quantitative methods use:
A. Only text analysis
B. Numerical data + math models
C. Purely qualitative interviews
D. Random guessing

**Answer:** B  
**Explanation:** Quantitative uses numbers + models.

---

**Question 2:** Categorical data example:
A. Age in years
B. Income in pesos
C. Approval status (Approved/Denied)
D. Temperature in Celsius

**Answer:** C  
**Explanation:** Categorical has discrete labels.

---

**Question 3:** Discrete numeric example:
A. Height in cm
B. Number of applicants
C. Weight in kg
D. Time in seconds

**Answer:** B  
**Explanation:** Discrete counts whole units.

---

**Question 4:** Overfitting means:
A. Model too simple
B. Memorizing noise, poor generalization
C. Perfect accuracy always
D. Ignoring training data

**Answer:** B  
**Explanation:** Overfitting learns noise.

---

**Question 5:** Python library for arrays:
A. Pandas
B. Matplotlib
C. NumPy
D. Flask

**Answer:** C  
**Explanation:** NumPy core for arrays.

# Quiz 2
## Scenario: Data Exploration
Rhea Joy validates modeling assumptions.

**Question 6:** Workflow step after data collection:
A. Deploy model immediately
B. Clean & explore
C. Ignore outliers
D. Skip validation

**Answer:** B  
**Explanation:** Clean/explore before modeling.

---

**Question 7:** Correlation vs causation:
A. Same thing
B. Correlation implies causation always
C. Correlation observes pattern, causation implies direct effect
D. Unrelated concepts

**Answer:** C  
**Explanation:** Correlation doesn't prove cause.

---

**Question 8:** Ratio scale has:
A. No meaningful zero
B. True zero allowing ratios
C. Only categories
D. Unordered labels

**Answer:** B  
**Explanation:** Ratio scale meaningful zero (can say "twice as much").

---

**Question 9:** Advantage of scripted analysis:
A. Harder to reproduce
B. Reproducibility & automation
C. Requires more manual work
D. Less scalable

**Answer:** B  
**Explanation:** Scripts enable reproducibility.

---

**Question 10:** Ethical concern in modeling:
A. Faster computation
B. Bias in training data
C. Prettier charts
D. Lower cost

**Answer:** B  
**Explanation:** Biased data propagates unfair predictions.
