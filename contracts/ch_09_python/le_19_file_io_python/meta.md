## Lesson 19 Exercises: File I/O Practice

---
### 1. Read File (6 minutes)
Create sample.txt with 3 lines. Read into list using readlines(); print each.

---
### 2. Write Names (6 minutes)
List names = ["Ana","Ben","Carla"]. Write to names.txt, one per line.

---
### 3. Append Log Entry (6 minutes)
Append timestamp + message to log.txt using mode "a".

---
### 4. Existence Check (6 minutes)
Function safe_read(path): check if exists; read if yes, return None if no.

---
### 5. Line-by-Line Processing (8 minutes)
Read file; strip whitespace; filter non-empty lines; collect in list.

---
### 6. Write List to File (7 minutes)
Given list of dicts (residents), write each name to file using writelines.

---
### 7. Encoding Practice (6 minutes)
Create file with Filipino text using encoding="utf-8"; read back and print.

---
### 8. Stretch: File Copy Utility (Optional)
Function copy_file(src, dest): read src, write to dest using binary mode.

---
### 9. Reflection (3 minutes)
Two file I/O mistakes to avoid.
