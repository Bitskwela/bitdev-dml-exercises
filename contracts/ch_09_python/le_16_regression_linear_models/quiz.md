# Quiz 1
## Scenario: Cost Prediction
Tian models scholarship expenses.

**Question 1:** Linear regression models:
A. Categorical outcomes
B. Relationship y ~ x
C. Random events only
D. Text similarity

**Answer:** B  
**Explanation:** Regression models dependent/independent relationship.

**Question 2:** Simple linear form:
A. y = mx + b
B. y = x^2
C. y = log(x)
D. y = e^x

**Answer:** A  
**Explanation:** y = mx + b linear equation.

**Question 3:** R-squared interprets:
A. Slope value
B. Fit quality (0-1)
C. Intercept
D. Data count

**Answer:** B  
**Explanation:** R^2 indicates proportion variance explained.

**Question 4:** Residuals are:
A. Predicted values
B. Actual - predicted
C. Independent variables
D. Slope coefficients

**Answer:** B  
**Explanation:** Residual = actual - predicted error.

**Question 5:** Good residual plot shows:
A. Clear upward trend
B. Random scatter around zero
C. Increasing variance
D. Perfect alignment

**Answer:** B  
**Explanation:** Random residuals suggest good fit.

# Quiz 2
## Scenario: Model Validation
Rhea Joy checks assumptions.

**Question 6:** Linearity assumption means:
A. All data points on line
B. Relationship approximately linear
C. No outliers exist
D. Only positive values

**Answer:** B  
**Explanation:** Linearity: relationship roughly straight.

**Question 7:** High R^2 guarantees:
A. Causation proven
B. Good fit within data range
C. Extrapolation always safe
D. No outliers present

**Answer:** B  
**Explanation:** High R^2 shows fit, not causation.

**Question 8:** Polynomial regression fits:
A. Straight line only
B. Curved relationships
C. Categorical data
D. Text patterns

**Answer:** B  
**Explanation:** Polynomial handles curves.

**Question 9:** scipy.stats.linregress returns:
A. Only slope
B. slope, intercept, r_value, p_value, std_err
C. Predictions array
D. Raw data

**Answer:** B  
**Explanation:** linregress comprehensive output.

**Question 10:** Prediction beyond data range:
A. Always accurate
B. Extrapolation, higher uncertainty
C. Guaranteed safe
D. Impossible

**Answer:** B  
**Explanation:** Extrapolation risks higher error.
