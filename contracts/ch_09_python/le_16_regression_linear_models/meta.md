## Lesson 16 Exercises: Regression Practice

---
### 1. Fit Linear Model (8 minutes)
x = [5,10,15,20,25], y = [12,23,35,45,56]. Use stats.linregress; print slope, intercept, R^2.

---
### 2. Prediction Function (6 minutes)
Define predict(x) using fitted parameters; predict for x=30.

---
### 3. Residual Calculation (7 minutes)
Compute residuals for each training point; list values.

---
### 4. Visualization (8 minutes)
Plot scatter of data + fitted line using matplotlib.

---
### 5. R-Squared Interpretation (6 minutes)
R^2 = 0.95. Explain what this means in context (2 sentences).

---
### 6. Bad Fit Detection (7 minutes)
Given x, y with clear curve; fit linear; compute low R^2; explain mismatch.

---
### 7. Extrapolation Warning (6 minutes)
Predict y for x=100 (far beyond training range); note uncertainty caveat.

---
### 8. Stretch: Polynomial Fit (Optional)
Use np.polyfit degree 2; compare R^2 to linear.

---
### 9. Reflection (3 minutes)
Two scenarios where linear regression inappropriate.
