## Lesson 17 Exercises: Simulation Practice

---
### 1. Random Integer Generation (5 minutes)
Generate list of 50 random integers 10-100.

---
### 2. Normal Distribution Sample (6 minutes)
Generate 200 values from normal (mean 50, std 10); compute sample mean, std.

---
### 3. Reproducible Simulation (6 minutes)
Set seed; generate 10 random numbers; reset seed; confirm same sequence.

---
### 4. Monte Carlo Budget (10 minutes)
Simulate 1000 scenarios: applicants ~ normal(100, 20), cost = applicants * 600. Compute mean, 95th percentile.

---

---
### 7. Histogram Visualization (7 minutes)
Plot histogram of Monte Carlo cost outcomes; add vertical line at mean.

---
