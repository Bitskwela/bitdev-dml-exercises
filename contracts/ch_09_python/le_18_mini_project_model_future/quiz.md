# Quiz 1
## Scenario: Forecast Validation
Tian reviews project components.

**Question 1:** Combining regression + simulation provides:
A. Trend only
B. Trend + uncertainty quantification
C. Random noise only
D. No forecast

**Answer:** B  
**Explanation:** Regression trend + simulation variability.

**Question 2:** Confidence interval interpretation:
A. Exact future value
B. Range likely containing true value
C. Guaranteed boundary
D. Mean only

**Answer:** B  
**Explanation:** CI gives probable range.

**Question 3:** Historical std used to:
A. Ignore variability
B. Model forecast uncertainty
C. Replace regression
D. Eliminate predictions

**Answer:** B  
**Explanation:** Std models natural variation.

**Question 4:** Visualization should include:
A. Only raw data
B. Data, forecast, CI
C. Nothing
D. Text only

**Answer:** B  
**Explanation:** Full viz shows context + uncertainty.

**Question 5:** Model limitation example:
A. Perfect accuracy
B. Short historical data, linear assumption
C. No limitations
D. Infinite precision

**Answer:** B  
**Explanation:** Short data, linearity assumptions are limits.

# Quiz 2
## Scenario: Presentation Prep
Rhea Joy prepares stakeholder report.

**Question 6:** Budget estimate derives from:
A. Forecast applicants * cost_per
B. Historical maximum
C. Random guess
D. No calculation

**Answer:** A  
**Explanation:** Budget = applicants * unit cost.

**Question 7:** Percentile(2.5) and (97.5) define:
A. Mean
B. 95% confidence interval
C. Median only
D. Mode

**Answer:** B  
**Explanation:** 2.5th, 97.5th percentiles = 95% CI.

**Question 8:** Linear trend assumption valid when:
A. Data shows curve
B. Relationship roughly straight
C. Random scatter
D. Cyclical pattern

**Answer:** B  
**Explanation:** Linear valid for approximately straight relationship.

**Question 9:** Project extension idea:
A. Ignore new data
B. Add seasonal or multi-variate factors
C. Remove all models
D. Stop forecasting

**Answer:** B  
**Explanation:** Extensions improve model realism.

**Question 10:** Report should highlight:
A. Only numbers
B. Forecast, CI, assumptions, limitations
C. Ignore uncertainty
D. Perfect confidence

**Answer:** B  
**Explanation:** Complete report includes assumptions, limits.
