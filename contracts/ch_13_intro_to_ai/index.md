# Prologue: Welcome to the Age of Intelligent Machines 🤖

From the carinderia order screens at lunch to the ride-hailing app that finds you a jeepney home, AI is already woven into daily Filipino life. This course takes you from "what even _is_ AI?" to deploying your own working AI app — no PhD required.

Follow **Dan Santos**, a curious learner from Diliman, as he discovers how machines learn, where they fail, and how to build with them responsibly. Every concept is grounded in scenarios you already know: Tita Malou's carinderia, GCash transactions, Grab rides, Shopee carts, and the everyday rhythms of barangay life.

---

## Course Structure: 25 Lessons Across 6 Modules

### **Module 01: Foundations of AI (Lessons 1-6)**

_Understanding What AI Really Is_

- L01: What is AI?
- L02: Narrow vs General AI
- L03: AI vs ML vs DL
- L04: How AI Works
- L05: Data in AI
- L06: Structured vs Unstructured Data

**Master:** AI definitions and history, the AI/ML/DL hierarchy, the role of data, the difference between rule-based systems and learning systems

---

### **Module 02: Python for AI (Lessons 7-9)**

_The Language of Modern AI_

- L07: Intro to Python for AI
- L08: Python Basics — Lists, Loops, and Decisions
- L09: Working with Libraries

**Master:** Python syntax essentials, control flow, working with `numpy` / `pandas`, and the library ecosystem AI engineers actually use

---

### **Module 03: How Machines Learn (Lessons 10-13)**

_From Algorithms to Deep Learning_

- L10: What is an Algorithm?
- L11: Heuristics vs Learning
- L12: Intro to Neural Networks
- L13: What is Deep Learning?

**Master:** Algorithmic thinking, the difference between hand-coded heuristics and learned models, neuron and layer fundamentals, why deep networks unlocked modern AI

---

### **Module 04: AI in the Real World (Lessons 14-18)**

_Where AI Lives Today_

- L14: Natural Language Processing
- L15: Computer Vision
- L16: Reinforcement Learning
- L17: Ethics in AI
- L18: AI Applications

**Master:** Core NLP and CV tasks, how RL agents learn from rewards, bias and fairness considerations, and a tour of AI in finance, health, agriculture, and government

---

### **Module 05: Working with LLMs and APIs (Lessons 19-21)**

_Talking to Modern AI Models_

- L19: Prompt Engineering
- L20: APIs in AI
- L21: OpenAI API Basics

**Master:** Effective prompting patterns, how AI APIs work over HTTP, authenticating and calling the OpenAI API safely (with mock fallbacks for the in-browser sandbox)

---

### **Module 06: Build and Ship an AI App (Lessons 22-25)**

_From Notebook to Production_

- L22: Build an LLM Chatbot (Part 1)
- L23: Build an LLM Chatbot (Part 2)
- L24: Improve Chatbot
- L25: Deploy AI App

**Master:** Designing a chat loop, conversation memory, refining responses with system prompts, and packaging an AI app for deployment

---

## What You'll Build

Across the chapter you'll build progressively richer AI projects:

1. **Rule-based "AI"** (L01) — your first decision system in pure Python
2. **Data classifier** (L05–L06) — sort structured and unstructured inputs
3. **Mini neural network intuition** (L12–L13) — see how layers transform data
4. **NLP and CV demos** (L14–L15) — text and image processing in Python
5. **Prompt-engineered helper** (L19) — turn vague asks into reliable outputs
6. **LLM Chatbot** (L22–L24) — a multi-turn assistant with memory
7. **Deployable AI app** (L25) — package and ship your chatbot

## Learning Approach

Each lesson features:

- **Story-driven content** — follow Dan Santos through relatable Filipino AI scenarios
- **Locally-themed examples** — carinderia menus, GCash fraud checks, Grab ETAs, Shopee recommendations, sinigang recipes
- **Hands-on Python practice** — every lesson ships starter code (`act_1.py`) and a worked solution (`act_1.answer.py`)
- **Self-check quizzes** — short knowledge checks per lesson
- **Sandbox-safe code** — OpenAI lessons include `MockOpenAI` fallbacks so you can run them without an API key
