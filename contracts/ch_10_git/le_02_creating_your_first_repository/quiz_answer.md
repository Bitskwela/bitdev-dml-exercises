# Quiz Answers: Creating Your First Repository

## Answer Key

### Question 1: What command initializes a new Git repository?

**Answer: B) `git init`**

### Question 2: What is stored in the `.git` directory?

**Answer: C) All version control information including commits, branches, and configuration**

### Question 3: Which command sets your name for Git commits?

**Answer: B) `git config user.name`**

### Question 4: What is the difference between global and local Git configuration?

**Answer: B) Global applies to all repositories, local applies only to the current repository**

### Question 5: What does `git status` display?

**Answer: C) The current branch, staged changes, unstaged changes, and untracked files**

### Question 6: True or False - Multiple repositories

**Answer: True**

## Common Misconceptions

1. **`.git` contains your files**: No, `.git` stores metadata about changes, not copies of your files
2. **`git init` can only be run once**: You can reinitialize, though it won't hurt the repository
3. **Configuration must be global**: Configuration can be local per repository, which is useful in team environments
