# Quiz: Viewing History

Answer the following questions to check your understanding of this lesson.

## Question 1

Which command displays the commit history of a repository?

A) `git history`
B) `git log`
C) `git show-commits`
D) `git commits`

**Correct Answer: B**

**Explanation:** The `git log` command displays the commit history, showing commit hashes, authors, dates, and commit messages.

---

## Question 2

What does `git log --oneline` do?

A) Shows only the first commit
B) Displays each commit on a single line with abbreviated hash and message
C) Shows commits from the past hour only

**Correct Answer: B**

**Explanation:** The `--oneline` flag condenses each commit to one line, showing the short hash and commit message for easier reading.

---

## Question 3

Which command shows the details of a specific commit, including the changes made?

A) `git show <commit-hash>`
B) `git view <commit-hash>`
C) `git details <commit-hash>`

**Correct Answer: A**

**Explanation:** The `git show` command displays detailed information about a specific commit, including the diff of changes made.

## Question 5

What is the purpose of `git log -- file.txt`?

A) To view the commit history for a specific file
B) To delete the file from history
C) To create a new file

**Correct Answer: A**

**Explanation:** Using `git log -- file.txt` shows the commit history for that specific file, helping you track when and how it changed.
