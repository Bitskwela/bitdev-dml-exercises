# Quiz: Lesson 46 - Frontend JS + Flask Deployment

---

# Quiz 1

### Question 1
**What is the difference between development and production environments?**
- A) Development uses Python, production uses JavaScript
- B) Development runs on localhost, production runs on a server
- C) There is no difference
- D) Development is slower

**Answer:** B) Development runs on localhost, production runs on a server

---

### Question 2
**What should DEBUG be set to in production?**
- A) True
- B) False
- C) 1
- D) "production"

**Answer:** B) False

---

### Question 3
**What is a Procfile used for?**
- A) To list Python dependencies
- B) To tell the platform how to run your app
- C) To store environment variables
- D) To configure the database

**Answer:** B) To tell the platform how to run your app

---

### Question 4
**What is gunicorn?**
- A) A database
- B) A production WSGI server for Python
- C) A frontend framework
- D) A testing tool

**Answer:** B) A production WSGI server for Python

---

### Question 5
**Which platform is recommended for deploying Flask apps?**
- A) WordPress
- B) Render
- C) Photoshop
- D) Excel

**Answer:** B) Render

---

### Question 6
**What database should you use in production instead of SQLite?**
- A) MySQL
- B) MongoDB
- C) PostgreSQL
- D) Redis

**Answer:** C) PostgreSQL

---

### Question 7
**What does CORS stand for?**
- A) Cross-Origin Resource Sharing
- B) Centralized Online Resource System
- C) Client-Origin Request Sharing
- D) Cached Object Resource Storage

**Answer:** A) Cross-Origin Resource Sharing

# Quiz 2

### Question 8
**Where do you set environment variables in Render?**
- A) In the code
- B) In the dashboard under Environment tab
- C) In a .txt file
- D) You don't need them

**Answer:** B) In the dashboard under Environment tab

---

### Question 9
**What file lists all Python dependencies?**
- A) dependencies.txt
- B) packages.txt
- C) requirements.txt
- D) modules.txt

**Answer:** C) requirements.txt

---

### Question 10
**How do you generate requirements.txt?**
- A) `pip list > requirements.txt`
- B) `pip freeze > requirements.txt`
- C) `python requirements.py`
- D) Manually write it

**Answer:** B) `pip freeze > requirements.txt`

---

### Question 11
**What protocol should production websites use?**
- A) HTTP
- B) FTP
- C) HTTPS
- D) SSH

**Answer:** C) HTTPS

---

### Question 12
**What is the purpose of a health check endpoint?**
- A) To check if users are healthy
- B) To monitor if the app and database are working
- C) To test user passwords
- D) To backup data

**Answer:** B) To monitor if the app and database are working

---

### Question 13
**Where can you view deployment logs in Render?**
- A) In the code
- B) In the Logs tab of the dashboard
- C) In your email
- D) In the database

**Answer:** B) In the Logs tab of the dashboard

---

### Question 14
**What does CDN stand for?**
- A) Content Delivery Network
- B) Central Data Node
- C) Client Database Network
- D) Cached Document Navigator

**Answer:** A) Content Delivery Network

---

### Question 15
**What should you do before deploying to production?**
- A) Delete all code
- B) Set DEBUG=False and use strong SECRET_KEY
- C) Remove the database
- D) Disable HTTPS

**Answer:** B) Set DEBUG=False and use strong SECRET_KEY
