# Activity: Building the Professional Barangay Portal Foundation

In this activity, you will step into the shoes of an **Elite Developer** to build the core foundation of the **Barangay Sto. Niño Official Portal**. Follow the tasks sequentially to construct a valid, pro-grade HTML5 skeleton in the editor.

---

## Task 1: The Modern Declaration
Every elite project starts by telling the browser exactly what it's dealing with. Declare the document as HTML5.

**Action:** Type the DOCTYPE declaration on the first line.
```html
<!DOCTYPE html>
```

---

## Task 2: The Root Container
Now, create the root `<html>` element that will contain your entire project. Don't forget to set the language to English (Philippines).

**Action:** Add the `html` tags with the `lang` attribute.
```html
<html lang="en-PH">
</html>
```

---

## Task 3: The Brain (Head)
Computers need instructions that users never see. Create the `<head>` section inside your `<html>` tags.

**Action:** Nest the `<head>` tags inside the `<html>` element.
```html
<head>
</head>
```

---

## Task 4: Character & Mobile Configuration
Elite developers ensure their sites work for everyone. Add meta tags for character encoding (so we can display the "ñ" in Sto. Niño) and mobile responsiveness.

**Action:** Add these two `<meta>` tags inside your `<head>`.
```html
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

---

## Task 5: Establish Identity (Title)
Your site needs a name in the browser tab. This is crucial for both users and Google.

**Action:** Add a `<title>` tag inside the `<head>`.
```html
<title>Barangay Sto. Niño | Official Portal</title>
```

---

## Task 6: The Visible Heart (Body)
Now, create the `<body>` section where all visible content will live. It should be placed after the `<head>`, still inside the `<html>` tags.

**Action:** Add the `<body>` tags after your `<head>`.
```html
<body>
</body>
```

---

## Task 7: The Masterpiece (Final Content)
Let's add some actual data to the portal. Inside the `<body>`, add a main heading and a short description.

**Action:** Add an `<h1>` and a `<p>` tag inside the `<body>`.
```html
<h1>Welcome to Barangay Sto. Niño</h1>
<p>The official digital home of our community.</p>
```

---

### Elite Developer Checklist:
- [ ] Is `<!DOCTYPE html>` on line 1?
- [ ] Are all tags properly nested (indented)?
- [ ] Is the `lang` attribute set to `en-PH`?
- [ ] Did you include the `UTF-8` charset?
