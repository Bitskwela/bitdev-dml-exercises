### **Course 01: Web Fundamentals (Lessons 1-5)**

- L01: What is the Internet?
- L02: How Websites Work
- L03: Frontend vs Backend
- L04: How Browsers Work
- L05: Dev Tools and Editors

**Master:** Internet basics, client-server architecture, browser rendering, developer environment setup

---

### **Course 02: HTML Foundations (Lessons 6-11)**
*Building the Structure of the Web*

- L06: HTML Document Structure
- L07: Tags, Elements, and Attributes

- L09: Links and Images
- L10: Tables, Forms, and Inputs

**Master:** HTML syntax, document structure, forms, semantic elements, accessibility basics

**Master:** CSS syntax, selectors, box model, layout fundamentals, positioning strategies

### **Course 04: Responsive Design (Lessons 18-22)**

### **Course 13: Full-Stack Web Apps (Lessons 37-42)**
