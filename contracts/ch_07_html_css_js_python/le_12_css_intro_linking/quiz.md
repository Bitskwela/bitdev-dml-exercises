# Quiz: Lesson 12 - What is CSS and How to Link It

---

# Quiz 1

### Question 1
**What does CSS stand for?**
- A) Computer Style Sheets
- B) Cascading Style Sheets
- C) Creative Style System
- D) Coded Style Syntax

**Answer:** B) Cascading Style Sheets

---

### Question 2
**What is the main purpose of CSS?**
- A) To structure web pages
- B) To add interactivity
- C) To style and design HTML elements
- D) To create databases

**Answer:** C) To style and design HTML elements

---

### Question 3
**Which analogy best describes the relationship between HTML and CSS?**
- A) HTML is the engine, CSS is the fuel
- B) HTML is the skeleton, CSS is the skin and clothing
- C) HTML is the database, CSS is the query
- D) HTML is the client, CSS is the server

**Answer:** B) HTML is the skeleton, CSS is the skin and clothing

---

### Question 4
**What is the correct syntax for a CSS rule?**
- A) `selector { property: value; }`
- B) `<selector property="value">`
- C) `selector = property: value`
- D) `property.selector(value)`

**Answer:** A) `selector { property: value; }`

---

### Question 5
**What are the three ways to add CSS to HTML?**
- A) Internal, External, Universal
- B) Inline, Internal, External
- C) Local, Global, Remote
- D) Static, Dynamic, Hybrid

**Answer:** B) Inline, Internal, External

---

### Question 6
**Which CSS method is considered BEST PRACTICE for most websites?**
- A) Inline CSS
- B) Internal CSS
- C) External CSS
- D) All are equally good

**Answer:** C) External CSS

---

### Question 7
**How do you link an external CSS file in HTML?**
- A) `<css src="style.css">`
- B) `<link rel="stylesheet" href="style.css">`
- C) `<style src="style.css">`
- D) `<include file="style.css">`

**Answer:** B) `<link rel="stylesheet" href="style.css">`

---

# Quiz 2

### Question 8
**Where should the `<link>` tag for CSS be placed?**
- A) In the `<body>` section
- B) In the `<head>` section
- C) At the end of the HTML file
- D) Anywhere in the document

**Answer:** B) In the `<head>` section

---

### Question 9
**What is inline CSS?**
- A) CSS written in a separate file
- B) CSS written inside `<style>` tags
- C) CSS written directly in HTML tags using `style=""` attribute
- D) CSS downloaded from the internet

**Answer:** C) CSS written directly in HTML tags using `style=""` attribute

---

### Question 10
**Which CSS method has the highest priority (overrides others)?**
- A) External CSS
- B) Internal CSS
- C) Inline CSS
- D) Browser default CSS

**Answer:** C) Inline CSS

---

### Question 11
**What does the "rel" attribute specify in a link tag?**
- A) The file path
- B) The relationship (that it's a stylesheet)
- C) The CSS version
- D) The priority level

**Answer:** B) The relationship (that it's a stylesheet)

---

### Question 12
**If you have CSS in a subfolder, which path is correct?**
- A) `href="style.css"`
- B) `href="css/style.css"`
- C) `href="/style.css"`
- D) `href="style/css.css"`

**Answer:** B) `href="css/style.css"`

---

### Question 13
**How do you write a comment in CSS?**
- A) `// This is a comment`
- B) `<!-- This is a comment -->`
- C) `/* This is a comment */`
- D) `# This is a comment`

**Answer:** C) `/* This is a comment */`

---

### Question 14
**What is a CSS reset used for?**
- A) To delete all CSS files
- B) To remove browser default styles for consistency
- C) To reset the page color to white
- D) To clear the browser cache

**Answer:** B) To remove browser default styles for consistency

---

### Question 15
**If multiple CSS files are linked, which one takes priority?**
- A) The first one
- B) The largest file
- C) The last one (loaded later)
- D) They all have equal priority

**Answer:** C) The last one (loaded later)
